## 🛡️ Principios de Diseño
1. **Valor Inmediato (Value-First):** Las métricas clave se exponen en el HERO, no en el footer.
2. **Estética de Ingeniería:** Dark mode severo (`#050505`), acentos precisos (`#2ecc71`), y tipografía JetBrains Mono para denotar CLI-first.
3. **Privacidad (Zero-Trust):** Purgado de historiales de proyectos descartados. Información estrictamente técnica y estoica.
4. **Dominio Bilingüe:** Interfaz construida para reflejar competencia nativa en Español e Idioma Profesional en Inglés.
