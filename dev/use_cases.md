# Casos de Uso (Use Cases)

## Visión General
El proyecto "CV-V2-Minimalist" tiene como objetivo posicionar a Mauro D. Asuaje G. como un Arquitecto de Experiencias Inteligentes y Senior Software Engineer. La plataforma funciona como un embudo de autoridad.

## Perfiles de Usuario Objetivo (Personas)
1. **El Reclutador Técnico (Tech Recruiter):** Busca evidencia rápida de habilidades (Tech Stack, años de experiencia, repositorios). Necesita poder descargar el CV en PDF en 1 clic.
2. **El Cliente B2B (Tech-Entrepreneur):** Busca un arquitecto de sistemas capaz de resolver problemas complejos (Automatización, RAG, Pipelines). Valora el diseño limpio, la sobriedad y la autoridad técnica.

## Casos de Uso Principales

### UC-01: Validación Inmediata de Autoridad
- **Actor:** Reclutador Técnico / Cliente B2B.
- **Acción:** Aterriza en la landing page.
- **Resultado Esperado:** Percibe un entorno de "ingeniería pura" (Dark mode minimalista). Identifica inmediatamente el rol (Growth Systems Engineer) y las fases de maestría.

### UC-02: Descarga de Evidencia Documental
- **Actor:** Reclutador Técnico.
- **Acción:** Hace clic en el Call to Action (CTA) `[DESCÁRGAME_CV.PDF]`.
- **Resultado Esperado:** El sistema sirve el archivo PDF más reciente sincronizado sin fricción.

### UC-03: Exploración de Laboratorios (Especialización)
- **Actor:** Cliente B2B.
- **Acción:** Lee la matriz de laboratorios/operaciones.
- **Resultado Esperado:** Comprende que el profesional no es un desarrollador genérico, sino un ingeniero que opera en áreas de alto valor (Document Intelligence, Growth Automation, Análisis Cuantitativo). El tono es sobrio y técnico, sin promesas vacías.
